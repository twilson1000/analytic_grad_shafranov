# analytic_grad_shafranov
Analytic description of Grad Shafranov equation.

We apply a method described in [Antoine J. Cerfon, Jeffrey P. Freidberg; “One size fits all” analytic solutions to the Grad–Shafranov equation. Phys. Plasmas 1 March 2010; 17 (3): 032502.](https://doi.org/10.1063/1.3328818).
